Server Requirements

Minimum Requirements

  • CPU: Intel i5-4460
  • RAM: 8GB
  • GPU: GTX 760
  • Storage: 120GB

Recommended Requirements

  • CPU: Intel i7-8700K
  • RAM: 16GB
  • GPU: RTX 2060
  • Storage: 250GB SSD

Required Software

  • GTA V (Legal Copy)
  • FiveM Client
  • Discord App
  • Teamspeak 3

Important Notes

  • Stable Internet Connection
  • Working Microphone
  • Age: 16+
  • English Communication

Server Guidelines

Server Rules

  • No RDM (Random Death Match)
  • No VDM (Vehicle Death Match)
  • No Breaking Character
  • Respect Other Players
  • No Exploiting or Hacking
  • Follow Staff Instructions

Roleplay Guidelines

  • Stay In Character At All Times
  • Use /me For Actions
  • Realistic Scenarios Only
  • No Powergaming
  • No Metagaming
  • Value Your Life (VL)

Important Information

  • Whitelist Required
  • Character Creation Approval
  • Regular Server Updates
  • Active Staff Support
  • Community Events
  • Discord Integration
